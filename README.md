# 💍 Rajarajeshwari & Padalingam — Wedding Invitation Website

A cinematic South Indian wedding invitation website built using pure HTML, CSS, and JavaScript.

Designed with:
- Luxury wedding aesthetics
- Smooth scroll animations
- Countdown timer
- Falling petals
- Music player
- Elegant gallery layout
- Responsive mobile-first design
